# Code of Conduct

## Our commitment

We want this project to be a respectful, inclusive, and practical space for collaboration.

## Expected behavior

- Be respectful and constructive
- Assume good intent and communicate clearly
- Focus feedback on code, docs, and outcomes
- Help keep discussions technical and actionable

## Unacceptable behavior

- Harassment, discrimination, or personal attacks
- Abusive, insulting, or demeaning language
- Threats, intimidation, or doxxing
- Repeated off-topic disruption or spam

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, and discussions.

## Enforcement

Maintainers may remove, edit, or reject comments/issues/PRs that violate this policy and may limit participation for repeated or severe violations.

## Reporting

If you experience or observe behavior that violates this Code of Conduct, report it through repository issues (or available private reporting channels), and include enough context for maintainers to act.
