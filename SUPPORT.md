# Support

## Support model

This is a best-effort open-source project maintained in spare time.

- There are no paid support plans.
- There are no guaranteed response times (no SLA).
- Community help is welcome and encouraged.

## Where to ask for help

For non-security help, please open a GitHub Issue and include enough context to reproduce the problem.

Recommended details:
- Host OS and version
- WSL version
- GPU type (NVIDIA / AMD / CPU-only)
- Docker and Compose versions
- Exact command(s) run
- Full error logs (sanitized if needed)

## What is in scope

- Setup and deployment issues for this repository
- Documentation clarifications and fixes
- Reproducible bugs in scripts, compose files, or documented flows

## What is out of scope

- Guaranteed troubleshooting turnaround
- General system administration outside this repo
- Third-party model support or legal interpretation of model licenses

## Security issues

Do not report vulnerabilities in public issues first.

See [SECURITY.md](SECURITY.md) for responsible disclosure and triage process.

## AI models and licenses

This repository does not create AI models.

Any model downloaded or used through these tools is a third-party artifact and is governed by its own license and terms of use. You are responsible for reviewing and complying with each model’s license before use, redistribution, or deployment.
